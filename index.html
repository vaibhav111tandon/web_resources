DEVPELOPMENT TOOLS

CODEPEN

CodePen is an online community for testing and showcasing user-created HTML, CSS and JavaScript code snippets.

JSFiddle

JSFiddle is an online community for testing and showcasing user-created and collaborational HTML, CSS and JavaScript code snippets, known as 'fiddles'.

BRACKETS

Brackets is a source code editor with a primary focus on web development. Created by Adobe Systems, it is free and open-source software licensed under the MIT License.

Visual Studio Code

Visual Studio Code is a source code editor. It includes support for debugging, embedded Git control, syntax highlighting, intelligent code completion, snippets, and code refactoring.

Sublime

Sublime Text is one of the most popular code editors available right now. It is adored by many programmers for it's speed, simplicity, and rich plugin ecosystem.

JS Bin

JS Bin is a tool for experimenting with web languages. In particular HTML, CSS and JavaScript, but JS Bin also supports other languages too (like Markdown, Jade and Sass).

W3 Schools

A reference guide to the many features of CSS.

Can I Use?

A quick, flexible and easy fluid grid for spectacularly easy responsive web design.

Clippy

A small tool to help you using the new and powerful clip-path property.

Excellent guide on how to write better, cleaner and more reusable CSS code.

Cubic-bezier function generator

A tool for visually generating timing animation functions to use in CSS transitions and animations.

CSS Easing Functions

An amazing collection of easing functions to be used in CSS transitions and animations.
